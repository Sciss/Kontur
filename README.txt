PREREQUISITES

Scala 2.8 snapshot / Java 1.6 (might work with 1.5).

Libraries:
- create a new folder "libraries"
- install into that folder
    ScissLib.jar (sourceforge.net/projects/scisslib)
    MRJAdapter.jar (https://mrjadapter.dev.java.net/)

Open the project in NetBeans 6.8
